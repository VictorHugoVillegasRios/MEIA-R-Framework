# Metodología MEIA-R  
**Modelo de Evaluación de Impacto Ambiental con Resiliencia Ecosistémica**

---

## 📌 Descripción
La metodología **MEIA‑R** representa un avance significativo en la Evaluación de Impacto Ambiental (EIA) al incorporar la **resiliencia ecosistémica** como modulador dinámico del impacto.  
Este enfoque permite una valoración más **precisa, realista y territorialmente diferenciada**, integrando ciencia ecológica con herramientas modernas de análisis de datos.

---

## 🎯 Objetivos
- 🔗 **Integrar variables clásicas con resiliencia ecosistémica** → ampliar la capacidad de análisis.  
- 📊 **Calcular impacto corregido** → clasificar impactos según resiliencia territorial.  
- 💻 **Modelo de datos optimizado** → reproducible, escalable y visualizable en Power BI.  

---

## 🚀 Innovación y Diferenciación
- La **EIA tradicional** mide impactos de forma estática.  
- La **MEIA‑R** incorpora resiliencia como variable moduladora.  
- Integra herramientas digitales modernas:  
  - 🐍 **Python** → cálculos reproducibles y automatizados.  
  - 💻 **Power BI** → dashboards interactivos para decisiones estratégicas.  
  - 🌍 **GIS** → análisis espacial y diferenciación territorial.  

---

## 📊 Ejemplo Práctico
- Impacto base: `I_Conesa = -86`  
- Resiliencia: `RE = 18`, `RE_norm = 0.75 (alta)`  
- Factor resiliencia: `F_resiliencia = 0.875` (reducción 12.5%)  
- Impacto corregido: `I_corregido = -75.25`  
- Clasificación: **Crítico → Severo**  
➡️ La alta resiliencia reduce el impacto en **12.5%**.
